# Приложение для прогноза погоды

Красноперова Елизавета, K3343

Веб-приложение для просмотра текущей погоды и прогноза на 3 дня. Работает с реальным API OpenWeatherMap, сохраняет данные в localStorage, поддерживает геолокацию и добавление до двух дополнительных городов.

## Функциональность

- **Геолокация при первом открытии** — автоматический запрос координат, показ погоды для текущего местоположения
- **Ручной ввод** — при отказе от геолокации или для поиска других городов
- **Прогноз на 3 дня** — отображается для каждого города (сегодня, завтра, послезавтра)
- **Дополнительные города** — можно добавить до 2 городов помимо основного
- **Автокомплит** — подсказки при вводе названия города (из заранее заданного списка)
- **Валидация** — проверка на дубликаты, существование города, лимит дополнительных городов, ошибки показываются под полем ввода
- **Кнопка обновления** — обновляет данные для всех городов без перезагрузки страницы
- **Сохранение состояния** — все города сохраняются в localStorage и восстанавливаются после перезагрузки
- **Адаптивный дизайн** — корректно отображается на мобильных и десктопах
- **Состояния интерфейса** — загрузка, ошибка, успех

## Технологии

- HTML5
- CSS3 (без фреймворков, кастомные стили, переменные, медиазапросы)
- JavaScript (без фреймворков)
- Хранение: localStorage
